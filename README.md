# Dowloader

Dowloader это программа для загрузки видео с открытым исходным кодом.

## Зависимости

- Консольная утилита [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- Библиотека [FFmpeg](https://github.com/FFmpeg/FFmpeg)

## Как установить данное ПО

1) Заходим в раздел [Releases](http://github.com/Leware56/Downloader/releases) и скачиваем zip
2) Заходим на FFmpeg и устанавливаем его с [официального сайта](https://ffmpeg.org)
3) После загрузки FFmpeg вам нужно переместить его в другое место и запомнить путь до папки `\bin` и вставить его в переменные среды в раздел `PATH`. Это нужно для того, чтобы могла работать функция "Только звук"
4) Можете пользоваться полноценным Downloader

## Как пользоваться Downloader

1) Вставьте вашу ссылку в главное поле и нажмите кнопку найти
2) Как только Downloader найдет ваш материал он отобразит время по середине между выбором качеством (слева) и флажком "только звук"
3) После этого нажимайте кнопку Download. Если в настройках стоит
